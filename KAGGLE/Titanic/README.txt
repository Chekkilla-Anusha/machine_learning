📌 Titanic Survival Prediction using Machine Learning
📖 Project Overview

This project aims to predict whether a passenger survived the Titanic disaster using machine learning techniques. The model is built using the Decision Tree algorithm and trained on the famous Kaggle Titanic dataset.

🎯 Objectives
Perform data preprocessing and cleaning
Handle missing values effectively
Select important features for prediction
Build a classification model using Decision Tree
Evaluate model performance
🛠️ Tech Stack
Python
Pandas
NumPy
Scikit-learn
Matplotlib
📂 Dataset
Dataset used: Titanic Dataset (Kaggle)
Contains information like:
Passenger class
Age
Gender
Fare
Survival status
⚙️ Steps Involved
Data Collection
Loaded dataset using Pandas
Data Preprocessing
Handled missing values
Converted categorical data into numerical format
Feature Selection
Selected important columns like Age, Sex, Pclass
Model Building
Applied Decision Tree Classifier
Model Evaluation
Checked accuracy and performance metrics
📊 Results
The model successfully predicts survival with good accuracy
Decision Tree helps in understanding decision rules clearly
